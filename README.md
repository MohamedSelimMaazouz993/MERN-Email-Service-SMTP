# Simple MERN-Email-Service-SMTP

## To Run this MERN App : 

You Need To Execute Server & Client in Different Terminal 

 For Server: 

```bash
  cd backend
  npm install 
  node server.js
```
For Client : 

```bash
  cd frontend
  npm install 
  npm start
```
